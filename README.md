# Life Organizer

**Life Organizer** is a C++ console application that helps you manage your daily life by tracking your budget and organizing your tasks efficiently.

---

## Features

### Budget Management
- Add, delete, and update **incomes** and **expenses**.
- View all incomes and expenses.
- Check your **current balance**.
- Generate **daily, weekly, monthly, and summary reports**.
- Calculate the percentage of expenses compared to income.

### Task Management
- Create multiple **task lists**.
- Add, delete, and update **tasks** within lists.
- View tasks by list or view **all tasks**.
- Categorize tasks by **priority** and **dates**.

### User Management
- Register a new user, login, and logout.
- User data is saved in a text file (`users.txt`).

---

## Requirements
- C++17 or later.
- Visual Studio or any C++ IDE.
- Git  

---

## How to Use
1. Open the project in Visual Studio.
2. Run the program (`F5` or **Start Debugging**).
3. You will see the welcome menu with options:
   - Register
   - Login
   - Budget Menu
   - Task Menu
   - Exit
4. Follow the prompts to add incomes, expenses, and create/manage tasks.

> **Note:** Dates are entered as `day month year`.
